Ejercicio I

Crea un programa en Java que permita registrar estudiantes en un curso. Cada estudiante tiene:

    Nombre (String)

    Edad (int)

    Nota final (double)

El programa debe:

    Usar una clase Estudiante para representar a cada estudiante.

    Pedir al usuario cuántos estudiantes desea ingresar (usar Scanner y un ciclo do-while para validar que el número sea mayor que 0).

    Usar un ciclo for para solicitar los datos de cada estudiante (nombre, edad, nota), crear objetos y agregarlos a un ArrayList.

    Mostrar un menú simple en consola (ciclo while) con las siguientes opciones:

        Mostrar todos los estudiantes

        Mostrar estudiantes aprobados (nota ≥ 6.0)

        Salir

        Conceptos aplicados
        Concepto	Aplicación en el ejercicio
        Declaración de variables	Al declarar nombre, edad, nota, etc.
        ArrayList	Para almacenar objetos Estudiante
        Clase	Estudiante con atributos y método mostrarInfo()
        for	Para ingresar estudiantes
        while	Para mostrar el menú principal
        do-while	Validar que el número de estudiantes sea mayor que 0

Entrega

Se debe entregar el link de github para su revisión.
